Interactive Demo

Try out different variable declarations and see how they behave. This interactive demo will help you understand the concepts better.

Variable Declaration Explorer

let declares a block-scoped variable that can be reassigned. 

let count = 1;
                      console.log(count); // 1

                      count = 2;
                      console.log(count); // 2

                      // Block scope demonstration
                      {
                        let blockVar = 'only in this block';
                        console.log(blockVar); // 'only in this block'
                      }
                      // console.log(blockVar); // ReferenceError

const declares a block-scoped variable that cannot be reassigned. 

const PI = 3.14159;
                      console.log(PI); // 3.14159

                      // PI = 3.14; // TypeError: Assignment to constant variable

                      // Objects declared with const can still be modified
                      const user = { name: 'John' };
                      user.name = 'Jane'; // This works!
                      console.log(user.name); // 'Jane'

                      // But you can't reassign the variable itself
                      // user = { name: 'Bob' }; // TypeError

var declares a function-scoped variable that can be reassigned and redeclared. 

var score = 100;
                  console.log(score); // 100

                  score = 200;
                  console.log(score); // 200

                  // Redeclaration is allowed with var
                  var score = 300;
                  console.log(score); // 300

                  // Function vs block scope demonstration
                  {
                    var blockVar = 'not actually block-scoped';
                  }
                  console.log(blockVar); // 'not actually block-scoped'
