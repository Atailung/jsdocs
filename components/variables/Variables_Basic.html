Variable Basics

Variables are containers for storing data values in JavaScript.

Why Variables Matter

  • Store and reuse data
  • Work with dynamic inputs
  • Track application state
  • Improve code readability

Think of variables as labeled boxes that store data.

JavaScript 
      
      // Creating a variable to store a user's name
      let userName = 'John';

      // Using the variable
      console.log('Hello, ' + userName + '!'); // Outputs: "Hello, John!"

      // Changing the value
      userName = 'Sarah';
      console.log('Hello, ' + userName + '!'); // Outputs: "Hello, Sarah!"
