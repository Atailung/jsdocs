Declaration Methods

JavaScript provides three ways to declare variables: let, const, and var. Each has different behaviors and use cases.

Recommended

let

Block-scoped, reassignable variable declaration.

  • Can be updated
  • Block-scoped
  • Cannot be redeclared in same scope
Immutable

const

Block-scoped, constant (non-reassignable) declaration.

  • Cannot be updated
  • Block-scoped
  • Cannot be redeclared
Legacy

var

Function-scoped, reassignable variable declaration.

  • Can be updated
  • Function-scoped (not block)
  • Can be redeclared
javascript
// Using let - can be reassigned
                  let age = 25;
                  age = 26; // Valid - value updated

                  // Using const - cannot be reassigned
                  const PI = 3.14159;
                  // PI = 3.14; // Error! Cannot reassign a constant

                  // Using var (older method)
                  var score = 100;
                  score = 110; // Valid - value updated
                  var score = 120; // Also valid - redeclared (not recommended)
